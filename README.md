Pryvate
=======

`pip install pryvate`

Private PyPi repository and proxy supporting:

* `python setup.py sdist upload...`
* `python setup.py sdist register...`
* Downloading private and public packages
* Configurable

Read more @ http://pryvate.readthedocs.org/en/latest/
